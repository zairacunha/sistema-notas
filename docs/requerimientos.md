# Requerimientos — Sistema de Notas
## Funcionales
- RF-01: El programa muestra un menú con opciones.
- RF-02: El usuario puede cargar un alumno con nombre y nota (1 a 10).
- RF-03: El usuario puede ver la lista de todos los alumnos con sus notas.
- RF-04: El programa calcula y muestra el promedio general del curso.
- RF-05: El programa muestra qué alumnos aprobaron (nota >= 6).
- RF-06: El programa muestra qué alumnos desaprobaron (nota < 6).
- RF-07: El usuario puede buscar un alumno por nombre.
- RF-08: El usuario puede salir del programa.
## No funcionales
- RNF-01: Los mensajes son claros y en español.
- RNF-02: Si el usuario ingresa una nota fuera de rango (menor a 1 o mayor a 10), muestra error.
- RNF-03: Si el usuario ingresa texto donde se espera un número, muestra error.
- RNF-04: El código está organizado en funciones separadas.
